# getRectangle
Processingで動かすOpenCVを用いた矩形を抽出するプログラム

### 応用
このプログラムを応用することで、このようなマス目の抽出ができるようになる。
ただし、画像処理なので当然抽出にミスることもあるので、もっとこんな手法がいいよ！っていうのがあれば是非教えて頂きたい。
